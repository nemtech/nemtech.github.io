    Account

    • トランザクションをアカウントから読み込む

      アカウント に関連する トランザクション のリストを取得します。

    • アカウントを作成して開く

      新しい アカウント を作成して、開きます。

    • アカウントへ送信したアセットの総量の取得

      あなたがアカウントへ送信したアセットの量を確認しましょう。

    • アカウントの情報を取得する

      公開鍵と アカウント の残高を取得します。

    • ネームスペースをアドレスへリンクする

      ネームスペース を アカウント にリンクしましょう。

    Account Restrictions

    • アカウント制限でスパム攻撃を防ぐ

      アカウント制限の追加・削除の方法を学ぶ

    Aggregate Transaction

    • アグリゲート完了トランザクションで支払いを送信する

      アグリゲート完了トランザクション を使用して、異なる複数のアカウントへトランザクションをアトミックに送ります。

    • アグリゲートボンドトランザクションを使ったエスクローの作成

      エスクローコントラクトの作成で アグリゲートボンドトランザクション について学習します。

    • アグリゲートボンドトランザクションを使ったモザイクの要求

      アグリゲートボンドトランザクション を使って、アカウントに資産を送ってもらうように問い合わせましょう。

    • アナウンス済みアグリゲートボンドトランザクションへの署名

      このガイドでは、あなたのアカウントの署名を要求するアグリゲートボンドトランザクションへ連署する方法を説明します。

    • アナウンス済みのアグリゲートボンドトランザクションへの署名を自動化する

      署名待ちのトランザクションへ自動的に署名する

    • マルチシグトランザクションの送信

      マルチシグ アカウントに関係するトランザクションを送ります。

    Block

    • 新しいブロックの監視

      新しい ブロック が取り込まれた通知を受け取る。

    • ブロック高によるブロックの取得

      与えられたブロック高の ブロック 情報を取得する。

    Cross-Chain Swaps

    • NEM パブリックとプライベート間のアトミッククロスチェーン交換

      クロスチェーンスワップ は、間に第三者の処理が入ることなく、異なるブロックチェーン間でのトークン交換を可能にします。

    Migration

    • NIS1 から Catapult へのマイグレーション

      Catapult では以前に利用可能だった NIS1 の機能のほとんどが進化しました。このドキュメントは以前の NIS1 の機能を、新たに利用可能な Catapult の技術に あなたのアプリケーションをアップグレード する手助けになるでしょう。

    Monitoring

    • 非同期なトランザクションのアナウンスを同期的に変える

      nem2-camel を使用して非同期なトランザクションのアナウンスを同期的に変えます。

    • トランザクションの監視

      アナウンスした後に トランザクション がブロックチェーンに含まれることを理解してください。

    Mosaic

    • モザイクの作成

      このガイドに従って モザイク を作成します。

    • モザイク供給量の変更

      このガイドではモザイクの供給量を変更します。

    • ネームスペースをモザイクへリンクする

      ネームスペース を モザイク にリンクしましょう。

    • モザイクの情報を取得する

      モザイク 識別子による所有権、可分性、期間、フラグの取得

    • レシート付きネームスペースの背後にあるアセットの識別子を取得

      レシートを使用して特定のエイリアスとトランザクションの解決を取得します。

    Multisig Account

    • アナウンス済みアグリゲートボンドトランザクションへの署名

      このガイドでは、あなたのアカウントの署名を要求するアグリゲートボンドトランザクションへ連署する方法を説明します。

    • マルチシグアカウントへの変換

      1-of-2 の マルチシグアカウント を作成します。

    • マルチシグアカウントの変更

      既存の マルチシグアカウント を変更します。

    • マルチレベルマルチシグアカウントの作成

      マルチレベル マルチシグアカウント を作成します。

    • マルチシグトランザクションの送信

      マルチシグ アカウントに関係するトランザクションを送ります。

    Namespace

    • ネームスペースの取得

      あなたの ネームスペース を取得しましょう。

    • サブネームスペースの作成

      このガイドに従って サブネームスペース を作成してください。

    • ネームスペースをモザイクへリンクする

      ネームスペース を モザイク にリンクしましょう。

    • ネームスペースをアドレスへリンクする

      ネームスペース を アカウント にリンクしましょう。

    • ネームスペースの情報を取得する

      ネームスペース 識別子による所有権と期間の取得

    • ネームスペース登録期限の延長

      ネームスペース 登録期限の延長。

    Network

    • ワークステーションのセットアップ

      最初のガイドでは、NEM で開発を始めるために必要なツールのインストールをステップバイステップに進めていきます。

    Receipt

    • レシート付きネームスペースの背後にあるアセットの識別子を取得

      レシートを使用して特定のエイリアスとトランザクションの解決を取得します。

    Transfer Transaction

    • 転送トランザクションの送信

      モザイク とメッセージを2つのアカウント間で転送します。

    • トランザクションの監視

      アナウンスした後に トランザクション がブロックチェーンに含まれることを理解してください。

    • 暗号化メッセージの送信

      受信者アカウントでしか読めない暗号化メッセージを送信します。
