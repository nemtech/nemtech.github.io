    • 新しいブロックの監視¶

    新しい ブロック が取り込まれた通知を受け取る。

    前提条件¶

    • 入門セクション を終了している

    それではコードを見ていきましょう¶

    View Code¶
    const listener = new Listener('http://localhost:3000');

listener.open().then(() => {

    listener
        .newBlock()
        .subscribe(block => console.log(block), err => console.error(err));

});

    View Code¶
    nem2-cli monitor block

    注釈

    クライアントサイドで使用する場合は(例, Angular, React, Vue) リスナーの実装は変わります 。

    お探しのものは見つかりましたか？ フィードバックをください。
