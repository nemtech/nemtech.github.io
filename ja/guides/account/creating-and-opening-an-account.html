    アカウントを作成して開く¶

    新しい アカウント を作成して、開きます。

    前提条件¶

    • 入門セクション を終了している
    • テキストエディタ または IDE
    • NEM2-SDK または CLI

    それではコードを見ていきましょう¶

    アカウントとは、NEM ブロックチェーン内に保存される変更可能な状態に関連付くキーペア(秘密鍵と公開鍵)です。

    View Code¶
    const account = Account.generateNewAccount(NetworkType.MIJIN_TEST);

console.log('Your new account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    const account = Account.generateNewAccount(NetworkType.MIJIN_TEST);

console.log('Your new account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    nem2-cli account generate --network MIJIN_TEST

    秘密鍵 は NEM アカウントを一意に特定し、全ての権限を保有します。オフライン のどこかで安全な場所にバックアップしていることが優先事項であり、誰にも共有してはいけません。

    公開鍵 は秘密鍵から暗号学的に算出され、共有しても安全です。しかし、ネットワークや正当性チェックなどの情報が含まれている アドレス を共有する方がより好ましいです。

    既に秘密鍵を持っている場合は、新しいアカウントを作成する必要はありません:

    View Code¶
    // Replace with a private key
const privateKey = process.env.PRIVATE_KEY as string;

const account = Account.createFromPrivateKey(privateKey, NetworkType.MIJIN_TEST);

console.log('Your account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
            // Replace with a private key
        final String privateKey = "";

        final Account account = Account.createFromPrivateKey(privateKey, NetworkType.MIJIN_TEST);
    View Code¶
    // Replace with a private key
const privateKey = process.env.PRIVATE_KEY;

const account = Account.createFromPrivateKey(privateKey, NetworkType.MIJIN_TEST);

console.log('Your account address is:', account.address.pretty(), 'and its private key', account.privateKey);

    ウォレットを使用する

    あなたの使用している SDK のプログラミング言語でクライアント側の開発が可能な場合は、ウォレットを開発することができます。

    ウォレットは、トランザクションへの署名をするため、パスワードで秘密鍵を暗号化して、アカウントを保存できるようにします。

    View Code¶
    const password = new Password('password');

const wallet = SimpleWallet.create('wallet-name', password, NetworkType.MIJIN_TEST);

const account = wallet.open(password);

console.log('Your new account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    const password = new Password('password');

const wallet = SimpleWallet.create('wallet-name', password, NetworkType.MIJIN_TEST);

const account = wallet.open(password);

console.log('Your new account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    nem2-cli account generate --network MIJIN_TEST --save --url http://localhost:3000 --profile test

    秘密鍵をお持ちですか？あなたの秘密鍵をインポートしてウォレットを作成することができます。

    View Code¶
    const password = new Password('password');

// Replace with a private key
const privateKey = process.env.PRIVATE_KEY as string;

const wallet = SimpleWallet.createFromPrivateKey('wallet-name', password, privateKey, NetworkType.MIJIN_TEST);

const account = wallet.open(password);

console.log('Your account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    const password = new Password('password');

// Replace with a private key
const privateKey = process.env.PRIVATE_KEY;

const wallet = SimpleWallet.createFromPrivateKey('wallet-name', password, privateKey, NetworkType.MIJIN_TEST);

const account = wallet.open(password);

console.log('Your account address is:', account.address.pretty(), 'and its private key', account.privateKey);
    View Code¶
    nem2-cli profile create --privatekey your_private_key --network MIJIN_TEST --url http://localhost:3000 --profile test
