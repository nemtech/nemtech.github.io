    • プロトタイピングツール¶

    注釈

    ⚠️ プロトタイピングツールは catapult-server 0.1 だけに対応しています。

    コーディング無しに NEM2 プロトタイピングツール を使ってブロックチェーンアプリケーションを作ります。

    NEM ノードをドラッグ＆ドロップして他のサービスとリンクし、プロトタイピングや学習のための新しく、楽しい方法を提供します。

    NEM2 Prototyping Tool は Node-RED ノードの集まりです。

    Node-RED はハードウェアやAPI、オンラインサービスと一緒に接続するための仮想的なプログラミングツールです。

    ノードを接続するブラウザベースのエディタを提供します。ノードのコンビネーションはクリックで実行できる名前付きのフローになります。

    Node-RED で作ったフローは、他の人と共有するためにエクスポート可能な JSON として保存されます。

    インストール¶

    Windows, Linux または Mac 向けの executable package をダウンロードして実行してください。

    すでに Node-RED はお持ちですか？ Manage palette を開いて node-red-contrib-nem2 パッケージをインストールしてください。

    Manage palette -> Install -> node-red-contrib-nem2

    ノードの設定¶

    左側のサイドバーは ノードパレット です。NEM に以下の関連したノードが確認できるでしょう:

    • NEM2 アカウント
    • NEM2 トランザクション
    • NEM2 リスナー
    • NEM2 ユーティリティ
    1. NEM account タブを開き、 account ノードをクリックしてください。

    右側のサイドバーはノードの説明、プロパティ、入力、出力を表示します。

    前のノードのアウトプットと続くノードのインプットへ繋げて、ノード同士をリンクさせてください。

    場合によっては、ノードをダブルクリックして、入力とプロパティを直接設定することもあります。

    ノードパレットとノードヘルプサイドバー

    1. 他のノードとアカウントをリンクしてみましょう。入力として privateKey を必要とするように、 NEM Account の下で、アカウント のノードと generateAccount をドラッグ＆ドロップし、リンクすることができます。

    ドラッグ&ドロップしてノード同士を接続します。

    1. アカウントノードをダブルクリックしてください。使用したい ネットワーク を選択してください。generateAccount ノードと同じにしてください。

    ノードのプロパティを完成させます。

    注釈

    アカウントノードのプロパティに private key があるのを発見しましたか？プロパティを設定すると、入力が上書きされます。つまり、空でない場合 generateAccount ノードによって提供される privateKey 出力の代わりに、このプロパティが使用されます。

    1. 最後に、アウトプット タブの下のパレットから debug ノードを見つけてください。これをアカウントへ接続して、 output のプロパティを complete msg object に変更してください。

    API ゲートウェイの設定¶

    いくつかのノードは API ゲートウェイの設定(例 announce transaction と listener ノード)が必要です。

    1. 設定が必要なノードをダブルクリックしてください。
    2. Server 入力フィールドの横にある鉛筆のアイコンをクリックしてください。
    3. http か https (例 http://localhost) スキーマの custom url を入力してください。あなたが使用したい network と port を選んで Update を押下してください。

    フローの実行¶

    1. 画面の右上にある deploy ボタンをクリックします。フローを実行するには、最初のノードに接続されている四角形を選択します。
    2. 右側のサイドバーのデバッグタブの下に出力が返されたことを確認します。NEM2 ノードは msg.,nem.<name_of_the_output> このような構造体を出力します。

    debug タブの下の出力を確認してください。

    フローのインポート¶

    1. Node-REDで Import > Clipboard from the menu (Ctrl-E) を選択してください。
    2. JSON フローをペーストして Import ボタンをクリックします。

    フローのエクスポート¶

    1. Node-RED でエクスポートしたいフローを選択します。
    2. メニュー (Ctrl-E) から Export > Clipboard を開き、ダイアログから JSON をコピーしてください。

    フローの共有¶

    私たちは私たちのコミュニティ - そう、あなた！ からの貢献をを喜んで受け入れます。あなたのフローをドキュメントへ含めるために、これらのガイドラインに従ってください。

    私たちはフローの共有に Node RED flows repository を使用しています。

    1. Add flow をクリック
    2. あなたのフローを端的でわかりやすいタイトルを、このフローは何をするのか、どうやって使うのかを説明してください。
    3. 共有したいフローの JSON をペーストしてください。
    4. NEM2 のタグを追加します。
