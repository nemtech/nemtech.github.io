ウォレット¶

デスクトップウォレット¶

アカウント、モザイク、名前空間を管理し、トランザクションを発行するための Symbol 用クロスプラットフォームクライアント。 Symbol Desktop Wallet は Linux, Mac, Windows と Web アプリケーションが提供されています。

インストール¶

  1. リリースセクション から Symbol Desktop Wallet インストーラを ダウンロード してください。

    各 OS 用の実行可能ファイルは アセット のセクションにあります。

  2. インストーラを 実行 して、手順に従います。

  3. 新しいアカウントを作成 するには ステップ・バイ・ステップガイド に従ってください。

ガイド¶

  • アカウントの作成

    トランザクションの送信と受信をするために、新しい Symbol アカウントを作成します。

  • ２つのアカウント間でモザイクとメッセージを送信

    定義して、署名したら転送トランザクションをアナウンスします。

  • ネームスペースの取得

    アセットを識別する、一意の名前を作成します。

  • モザイクの作成

    モザイクを使用した資産のトークン化

  • サブネームスペースの作成

    ルート ネームスペース を取得すると 3 つまでのサブネームスペースを作って、アセットの整理をすることができます。

  • モザイク供給量の変更

    「Supply Mutable」 プロパティを使用してモザイクを作成した場合、トークンを増やすか、総供給量を減らすことができます。

  • ネームスペースをアドレスへリンクする

    トランザクションを発行するとき、他の人がよりわかりやすい方法でアカウントを参照できるように、ネームスペースをアドレスにエイリアスとして設定します。

  • ネームスペースをモザイクへリンクする

    トランザクションを発行するとき、他の人がよりわかりやすい方法でモザイクを参照できるように、ネームスペースをモザイクにエイリアスとして設定します。

  • ネームスペース登録期限の延長

    このガイドでは、ネームスペース期限の延長の仕方を示します。

  • ロンチ前のオプトインから XYM を取得する

    このガイドでは、 Symbol ロンチ前 (2021年3月12日) のオプトインプロセスに登録した場合に、あなたにへ授与された XYM (Symbol の基軸通貨) へのアクセス方法を説明します。

  • Retrieving your XYM from a post-launch opt-in

    This guide explains how to enroll in the opt-in process once Symbol has launched (this is, after March 12, 2021).
