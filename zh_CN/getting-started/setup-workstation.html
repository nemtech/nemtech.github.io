    设置工作站¶

    这个指南将带领你把在NEM上开发所需的工具逐步地安装好。

    Catapult（投石车）服务引导程序的运作¶

    投石车服务器节点 (第一层） 组成点对点的区块链网络。投石车REST节点 (第二层）提供可以让应用程序和区块链及其功能接驳的接口网关（API Gateway)。

    您将使用 Catapult Service Bootstrap 运行私链来学习。 此服务在本地运行投石车服务器实例和投石车 REST节点。

    1. 在运行以下命令之前，docker 和 docker-compose：
    $> git clone https://github.com/tech-bureau/catapult-service-bootstrap.git
$> cd catapult-service-bootstrap
$> docker-compose up

    注解

    catapult service bootstrap不起作用吗？ 请查看 these troubleshooting tips。

    1. 检查您是否可以获得首区块的信息：
    $> curl localhost:3000/block/1

    获得测试账户¶

    帐户是与存储在NEM区块链中的可变状态相关联的密钥对（私钥和公钥）。 换句话说，您在区块链上有一个存款箱，只有您可以使用密钥对进行修改。 顾名思义，私钥必须始终保密。 得到私钥的任何人都可以控制该帐户。

    公钥 以密码方式导源于私钥。 要逆转这过程需要数百万年才能完成，因此公钥可以安全地共享。

    最后，遵循NEM区块链的步骤，以用公钥生成帐户地址。 分享地址比分享公钥好，因为它包含更多信息，例如有效性检查或它使用的网络（公链，测试链或私链）。

    NEM2-CLI 允许您方便地执行在终端中最常用的命令，即用它与区块链交互，设置账户，发送资金等。

    1. 使用 npm 下载NEM2-CLI。
    $> sudo npm install --global nem2-cli
    1. 打开终端，然后转到下载了Catapult Bootstrap Service的目录。
    $> cd  build/generated-addresses/
$> cat addresses.yaml
    1. 在 nemesis_addresses 的部分， 您会看到包含 cat.currency 的密钥对。区块链上的所有行动都需要cat.currency，以作验证和镇守网络的犒赏。
    2. 将第一个账户作为用户资料加载于NEM2-CLI中。
    $> nem2-cli profile create

Introduce network type (MIJIN_TEST, MIJIN, MAIN_NET, TEST_NET): MIJIN_TEST
Introduce your private key: 41************************************************************FF
Introduce NEM 2 Node URL. (Example: http://localhost:3000): http://localhost:3000
Insert profile name (blank means default and it could overwrite the previous profile):

    您应该可以在终端中看到以下字行，其中包含帐户凭据：

    New Account:    SCVG35-ZSPMYP-L2POZQ-JGSVEG-RYOJ3V-BNIU3U-N2E6

Public Key:     33E0...6ED

Private Key:    41....FF

    设置开发环境¶

    是时候选择编程语言了。 选择您最熟练的，或者按照您的项目要求进行选择。

    为新项目创建一个文件夹，然后运行所选语言的说明。

    Typescript
    Javascript
    1. 创建一个 package.json 文件。 Node.js所需的最低版本是8.9.X.
    $> npm init
    1. 安装nem2-sdk和rxjs库。
    $> npm install nem2-sdk rxjs
    1. nem2-sdk 是以 TypeScript 创建的。当给NEM区块链创建应用程式时，建议用 TypeScript。

    确保至少安装了2.5.X版。

    $> sudo npm install --global typescript
$> typescript --version
    1. 使用 ts-node 来执行带节点的TypeScript文件。
    $> sudo npm install --global ts-node
    1. 创建一个 package.json 文件。 Node.js所需的最低版本是8.9.X.
    $> npm init
    1. 安装nem2-sdk和rxjs库。
    $> npm install nem2-sdk rxjs

    继续: 编写你的第一个应用程序。
