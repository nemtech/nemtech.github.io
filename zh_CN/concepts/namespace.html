    命名空间¶

    命名空间允许您在NEM区块链上为您的企业和资产开辟一个链上的 独特空间 。

    A namespace starts with a name that you choose, similar to an internet domain name. If one account creates a namespace, that will appear as unique in the NEM ecosystem.

    帐户可以将注册名称（命名空间或子命名空间）与 账户 或 马赛克 标识相连接。

    子命名空间¶

    名在互联网上，域明可以具有子域名。 在NEM中，命名空间可以有子命名空间。

    您可以在不同的根命名空间创建多个同名子命名空间。比如，您可以创建 foo.bar 和 foo2.bar 子命名空间。

    命名空间最多 3 个层次，包含主命名空间和两层子命名空间。

    别名¶

    别名交易 让很长的地址变得好记，也可以让马赛克更有识别性。

    命名空间的创建者可以编辑命名空间和资产之间的联系。之后，特定交易的别名关系可以从区块的 收据 中复取。

    限制：

    • 一个帐户只能把名称与一个帐户或马赛克相关联，但这些帐户可以链接多个别名。
    • 账户能为其他可接受 AddressNamespaceTransactions 的账户指定名称。 但是，如果帐户想要将别名分配给马赛克ID (mosaicId），那么这个账户必须是马赛克的创建者。

    例子¶

    一位消费者购买了一张活动门票。票务公司把票发到了消费者的账户里。

    可识别的马赛克和地址

    区别发件人¶

    票务公司事先注册好 ticketsales 命名空间 ，并将其作为别名链接到它的账户。消费者可以迅速识别从票务公司账户传来的交易。

    区别门票¶

    同一家公司出售在不同场地举办的活动门票。该公司在每次开始售卖门票时都注册一个不可转让的 马赛克（mosaic）。

    票务公司在根域名 ticketsales 上添加了一系列子域名。根域名加子域名为``ticketsales.event<ID>.ticket``。

    该公司将一个已注册的马赛克与 ticketsales.event1.ticket 命名空间相链接。

    区别买家¶

    想要购买门票的Alice已经注册了命名空间 alice 并将其作为别名分配给她的帐户。

    票务公司可以将 一个 ticketsales.event1.ticket 发送给 alice 而不是把一个 0dc67fbe1cad29e3 发送给 SCVG35-ZSPMYP-L2POZQ-JGSVEG-RYOJ3V-BNIU3U-N2E6。

    模式¶

    注解

    配置参数是 可编辑的 。但在公共网络配置中可能有所不同。

    RegisterNamespaceTransaction¶

    您可以通过发布注册命名空间交易来进行注册或重新租用命名空间。

    版本: 0x02

    实体类型: 0x414E

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    namespaceType NamespaceType 已注册命名空间的类型。
    duration uint64 租借期限表示您想要租借该命名空间的时间长短，以已确认区块数量为单位。租借期间，可以通过发送 注册命名空间交易 来延长已确认区块数量的单位。当租借期完毕，命名空间将变为冻结状态。
    parentId uint64 子域名需要父命名空间的引用。
    namespaceId uint64 命名空间的id。
    namespaceNameSize uint8 命名空间名称大小。
    name array(bytes, namespaceNameSize) 命名空间的名字必须是独特的，且长度不可超过 64 字符。允许包含的字符有 a, b, c, …, z, 0, 1, 2, …, 9, ‘, _ , -。

    AddressAliasTransaction¶

    公布别名交易以将命名空间附加到帐户。 命名空间可以分配给网络中的任何帐户。

    版本: 0x01

    实体类型: 0x424E

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    aliasAction AliasAction 别名动作。
    namespaceId uint64 将成为别名的命名空间id。
    address 25字节(二进制) 有别名的地址。

    MosaicAliasTransaction¶

    公布别名交易把命名空间附加到马赛克上。要为马赛克设置别名，公布交易的帐户必须是创建相关命名空间和马赛克的账户。

    版本: 0x01

    实体类型: 0x434E

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    aliasAction AliasAction 别名动作。
    namespaceId uint64 将成为别名的命名空间id。
    mosaicId uint64 被附加别名的马赛克id。

    NamespaceType¶

    例举: uint8

    Id 说明
    0 根命名空间。
    1 子命名空间。

    别名动作¶

    例举: uint8

    Id 说明
    0 连接别名。
    1 取消连接别名。
