    Account Filter¶

    在特定的约束下，账户 可以配置一组智能条规以拦截或接受交易。

    可编辑的链上约束称为过滤器。 帐户可以配置以下类型：

    地址过滤器¶

    帐户可以决定仅从允许的 地址 列表中接收交易。同样地，帐户也可以指定不希望从中接收交易的地址列表。

    地址过滤器示意图

    注解

    允许和禁止过滤器是互斥的。 换句话说，帐户只能为每种类型的过滤器配置禁止或允许列表。

    默认情况下，如果没有设置过滤器，则网络中的所有帐户都可以向该帐户发起交易。

    马赛克过滤器¶

    只有在允许所有附加的 马赛克 的情况下，帐户才能配置过滤器以允许传入的交易。 另一方面，该帐户可以拒绝接受包含在已阻止列表中的马赛克的交易。

    实体类型过滤器¶

    账户可以允许或禁止公布 ·确定类型`的交易。这样做不但增加自身的安全性，防止错误地公布交易。

    例子¶

    阻止垃圾交易¶

    使用公链来验证其产品的质量。

    当质量验证过程结束时，操作员发送一个 质量印章 ·到产品账户。

    最终客户可以通过扫描二维码检阅产品马赛克。 因此，该公司只需显示相关交易，避免其他人发送带有与产品无关信息。

    阻止垃圾交易

    该公司选择配置他们的产品帐户过滤器，只允许接收包含 company.quality.seal 马赛克的交易。

    增强账户安全¶

    最近，爱丽丝只使用她的主要帐户签署一个 多重签名账户 的聚合交易。她是该多重签名账户的共签者。

    作为一项临时措施，爱丽丝选择禁止她的主账户公布转让交易，仔细检查她所拥有的任何资金是否会被转移。

    模式¶

    注解

    配置参数是 可编辑的。 公共网络配置可能有所不同。

    AccountPropertiesAddressTransaction¶

    Configure filters to prevent receiving transactions from undesired addresses.

    版本: 0x01

    实体类型: 0x4150

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    propertyType PropertyType 属性类型
    modificationsCount uint8 修改次数
    modifications array(AddressModification, modificationsCount) 一系列的修改

    AccountPropertiesMosaicTransaction¶

    配置过滤器以防止接收包含特定马赛克的交易。

    版本: 0x01

    实体类型: 0x4250

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    propertyType PropertyType 属性类型
    modificationsCount uint8 修改次数
    modifications array(MosaicModification, modificationsCount) 一系列的修改

    AccountPropertiesEntityTypeTransaction¶

    配置过滤器以防止公布特定 类型 的交易。

    版本: 0x01

    实体类型: 0x4350

    内联:

    • Transaction or EmbeddedTransaction
    属性 类型 说明
    propertyType PropertyType 属性类型
    modificationsCount uint8 修改次数
    modifications array(EntityTypeModification, modificationsCount) 一系列的修改

    AddressModification¶

    内联:

    • AccountPropertiesModification
    属性 类型 说明
    value 25字节(二进制) 被允许或被阻止的地址。

    MosaicModification¶

    内联:

    • AccountPropertiesModification
    属性 类型 说明
    value uint64 被允许或被阻止的马赛克ID

    EntityTypeModification¶

    内联:

    • AccountPropertiesModification
    属性 类型 说明
    value uint16 被允许或被阻止的 实体类型。

    AccountPropertiesModification¶

    属性 类型 说明
    modificationType PropertyModificationType 修改类型。

    PropertyType¶

    Enumeration: uint8

    Id 说明
    0x01 属性类型是一个地址。
    0x02 属性类型是一个马赛克id
    0x04 属性类型是一个交易类型
    0x05 属性类型哨兵。
    0x80 + type 该属性被解释为阻止型操作。

    PropertyModificationType¶

    Enumeration: uint8

    Id 说明
    0x00 添加属性值。
    0x01 删除属性值。
